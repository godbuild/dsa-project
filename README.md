# Mentorship App Frontend

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Set API URL (if needed):
   ```
   VITE_API_URL=http://localhost:5000/api
   ```
3. Run in development:
   ```
   npm run dev
   ```

## Testing

```
npm run test
```

## Deployment

Deploy with Vercel, Netlify, or your preferred static host.
